# 仿写美团M站

### 技术栈
- JavaScript
- zepto
- gulp
- PubSub(订阅发布者模式插件)
- swiper
- iscroll


### 首页
完成首页页面数据渲染，使用iscroll监听上拉加载更多和下拉刷新，菜单栏目使用swiper插件完成切换的效果

### 商品详情页面
点击一个商家店铺进入到对应的店铺请求数据，渲染页面，商品滚动左侧分类梯子功能，包括商品下单的价格数量的计算，下单跳转到订单详情页面

### 订单详情页面
显示购买商品的订单详情，商品列表渲染，价格计算，活动提示，地址定位
